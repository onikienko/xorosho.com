Поиск

rutracker.org

Расширение может встраивать на страницы xorosho.com ссылки для поиска релиза на rutracker.org (вход на rutracker должен быть выполнен)
Работает только для релизов в категориях "хорошая музыка" и "хорошее кино".


soundcloud.com (NEW)

Расширение может попытаться найти исполнителя на soundcloud.com. Только для категории "Хорошая музыка".


Информация

Корректная работа поиска зависит от соответсвия заголовка новости опрделенному шаблону. Для большинства новостей все будет работать. В крайнем случае, можно будет откорректировать поисковый запрос уже на самом сайте.

Так это будет выглядеть:
Проверка новостей Расширение будет периодически проверять новости на xorosho.com. При появлении новых, на значке расширения появится отметка. Также можно включить звуковое оповещение.
